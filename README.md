# 📋 Облік контролів та протокольних рішень

Веб-застосунок для обліку та контролю виконання наказів, заходів та протокольних рішень з можливістю зберігання даних у браузері (IndexedDB).

## 🚀 Особливості

- **Локальне зберігання даних** — використання Dexie.js (обгортка над IndexedDB) для зберігання даних у браузері
- **Повноцінний CRUD** — додавання, редагування, видалення записів
- **Фільтрація** — пошук за назвою, номером наказу, відповідальним; фільтр за статусом та напрямком
- **Статистика** — візуалізація кількості записів за статусами та напрямками
- **Прикріплення PDF** — можливість додавати PDF-файли до записів
- **Експорт даних** — вивантаження даних у Excel (.xlsx) формат
- **Налаштування** — керування відповідальними, напрямками, періодичністю та статусами через інтерфейс
- **Адаптивний дизайн** — зручне відображення на різних пристроях

## 📦 Технології

- **HTML5 / CSS3** — структура та стилі
- **JavaScript (ES6+)** — логіка застосунку
- **Dexie.js** — робота з IndexedDB
- **SheetJS (XLSX)** — експорт у Excel

## 📖 Використання

### Додавання запису
1. Заповніть форму:
   - Назва та номер наказу
   - Дата наказу
   - Заходи для виконання
   - Періодичність виконання
   - Термін виконання
   - Відповідальний
   - Напрямок (при виборі "Інше" з'явиться додаткове поле)
   - Статус
2. За бажанням додайте PDF-файл
3. Натисніть "Додати запис"

### Редагування
1. Натисніть кнопку ✏️ у рядку таблиці
2. Відредагуйте дані у формі
3. Натисніть "Оновити запис"

### Видалення
1. Натисніть кнопку 🗑️ у рядку таблиці
2. Підтвердіть видалення

### Фільтрація
- Використовуйте поле 🔍 Пошук для пошуку за текстом
- Оберіть статус у випадаючому списку для фільтрації
- Натискайте на статистику для швидкої фільтрації

### Налаштування
Натисніть кнопку ⚙️ у верхньому правому куті для:
- Керування відповідальними
- Керування напрямками
- Керування періодичністю
- Керування статусами

### Експорт даних
Натисніть кнопку 📊 Excel для вивантаження поточного відфільтрованого списку у .xlsx формат

## 📊 Структура даних

Кожен запис містить:

| Поле | Тип | Опис |
|------|-----|------|
| id | number | Автоінкрементний ідентифікатор |
| orderName | string | Назва наказу |
| orderNumber | string | Номер наказу |
| orderDate | string | Дата наказу (YYYY-MM-DD) |
| measures | string | Заходи для виконання |
| periodicity | string | Періодичність |
| deadline | string | Термін виконання (YYYY-MM-DD) |
| responsible | string | Відповідальна особа |
| direction | string | Напрямок діяльності |
| customDirection | string | Власний напрямок (якщо вибрано "Інше") |
| status | string | Статус виконання |
| pdfBlob | Blob | PDF-файл (зберігається у IndexedDB) |
| pdfName | string | Назва PDF-файлу |

## 🔧 Особливості localStorage

Всі налаштування зберігаються у localStorage браузера:
- `controls_responsible_persons` — список відповідальних
- `controls_directions` — список напрямків
- `controls_periodicity` — список періодичностей
- `controls_status_labels` — назви статусів
