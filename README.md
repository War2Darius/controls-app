## ✨ Нове у версії 2.0.0

- **🏷️ Налаштування з вкладками** — повністю перероблене модальне вікно з 5 вкладками:
  - 👥 Відповідальні
  - 📂 Напрямки
  - 🔄 Періодичність
  - 🏷️ Статуси
  - 📤 Експорт/Імпорт
- **🔒 Захист від XSS** — екранування HTML для всіх динамічних даних
- **📅 Валідація дат** — перевірка, що дата наказу не пізніша за термін виконання
- **📥 Виправлений імпорт** — коректна робота імпорту JSON та CSV з вибором режиму
- **❌ Закриття по Escape** — модальне вікно закривається клавішею Esc
- **🎨 Покращений дизайн** — кнопка закриття ✕, анімації, сповіщення
- **📱 Адаптивність** — покращено відображення на мобільних пристроях

## 🚀 Особливості

- **Локальне зберігання даних** — використання Dexie.js (обгортка над IndexedDB) для зберігання даних у браузері
- **Повноцінний CRUD** — додавання, редагування, видалення записів
- **Фільтрація** — пошук за назвою, номером наказу, відповідальним; фільтр за статусом та напрямком
- **Статистика** — візуалізація кількості записів за статусами та напрямками (клікабельна)
- **Прикріплення PDF** — можливість додавати PDF-файли до записів та переглядати їх
- **Експорт/Імпорт даних** — вивантаження та завантаження даних у форматах JSON та CSV
- **Налаштування** — керування відповідальними, напрямками, періодичністю та статусами через інтерфейс з вкладками
- **Адаптивний дизайн** — зручне відображення на різних пристроях

## 📦 Технології

- **HTML5 / CSS3** — структура та стилі
- **JavaScript (ES6+)** — логіка застосунку
- **Dexie.js** — робота з IndexedDB

## 📖 Використання

### Додавання запису

1. Заповніть форму:
   - Назва та номер наказу
   - Дата наказу
   - Заходи для виконання
   - Періодичність виконання
   - Термін виконання
   - Відповідальний (з випадаючого списку)
   - Напрямок (при виборі "Інше" з'явиться додаткове поле)
   - Статус
2. За бажанням додайте PDF-файл
3. Натисніть "Додати запис"

### Редагування

1. Натисніть кнопку ✏️ у рядку таблиці
2. Відредагуйте дані у формі
3. Натисніть "Оновити запис"

### Видалення

1. Натисніть кнопку 🗑️ у рядку таблиці
2. Підтвердіть видалення

### Фільтрація

1. Використовуйте поле 🔍 Пошук для пошуку за текстом
2. Оберіть статус у випадаючому списку для фільтрації
3. Натискайте на елементи статистики для швидкої фільтрації

### Налаштування

Натисніть кнопку ⚙️ у верхньому правому куті для відкриття модального вікна з вкладками:

| Вкладка | Функціонал |
|---------|------------|
| 👥 Відповідальні | Додавання/видалення відповідальних осіб |
| 📂 Напрямки | Додавання/видалення/редагування напрямків |
| 🔄 Періодичність | Додавання/видалення/редагування періодичності |
| 🏷️ Статуси | Редагування назв статусів |
| 📤 Експорт/Імпорт | Експорт/імпорт даних у JSON та CSV |

### Експорт/Імпорт даних

- **Експорт** — вивантаження даних у форматі JSON або CSV
- **Імпорт** — завантаження даних з вибором режиму:
  - 🔁 Замінити всі дані
  - 🔄 Об'єднати з поточними (тільки нові записи)
  - ➕ Додати всі (з можливими дублікатами)

## 📊 Структура даних

Кожен запис містить:

| Поле | Тип | Опис |
|------|-----|------|
| id | number | Автоінкрементний ідентифікатор |
| orderName | string | Назва наказу |
| orderNumber | string | Номер наказу |
| orderDate | string | Дата наказу (YYYY-MM-DD) |
| measures | string | Заходи для виконання |
| periodicity | string | Періодичність (ключ зі списку) |
| deadline | string | Термін виконання (YYYY-MM-DD) |
| responsible | string | Відповідальна особа |
| direction | string | Напрямок діяльності |
| customDirection | string | Власний напрямок (якщо вибрано "Інше") |
| status | string | Статус виконання (pending/in-progress/completed) |
| pdfBlob | Blob | PDF-файл (зберігається у IndexedDB) |
| pdfName | string | Назва PDF-файлу |

## 🔧 Особливості localStorage

| Ключ | Опис |

| controls_directions | Список напрямків |
